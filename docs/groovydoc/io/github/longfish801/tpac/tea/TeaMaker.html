Package: io.github.longfish801.tpac.tea

[Groovy] Trait TeaMaker

    • io.github.longfish801.tpac.tea.TeaMaker

    • tpac記法の解析にともない、各要素を生成する特性です。
      本クラスのメソッドは TeaPartyでの解析時に呼ばれます。
      独自DSLのため実装するときは newTeaDec(String,String)、 newTeaHandle(String,String,TeaHandle)を オーバーライドしてください。
