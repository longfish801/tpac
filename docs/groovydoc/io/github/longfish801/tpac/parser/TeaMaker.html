Package: io.github.longfish801.tpac.parser

[Groovy] Trait TeaMaker

    • io.github.longfish801.tpac.parser.TeaMaker

    • tpac記法の解析にともない、各要素を生成する特性です。
      tpac文書の読込は TeaServer.soakを利用してください。
      本クラスのメソッドは TeaPartyでの解析時に呼ばれます。
