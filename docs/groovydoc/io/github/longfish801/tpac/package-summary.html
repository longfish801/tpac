Package io.github.longfish801.tpac Description

tpac文書を扱うためのパッケージです。
author:
io.github.longfish801
  • Class Summary 
    Class Description
    TpacConst 固定値を保持します。
    TpacDec tpac文書の宣言です。
    TpacEval tpac文書のGroovy評価値です。
    TpacHandle tpac文書のハンドルです。
    TpacMaker tpac記法の解析にともない、各要素を生成します。
    TpacMsg メッセージを保持します。
    TpacParseExceptions.ParseError 解析エラーです。
    TpacParty tpac記法を解析します。
    TpacRefer tpac文書の参照です。
    TpacServer tpac文書を保持します。
  • Exception Summary 
    Exception Description
    TpacHandlingException tpac文書の操作に失敗したことを表す例外クラスです。
    TpacParseExceptions tpac文書の解析エラーのリストを保持する例外クラスです。
    TpacSemanticException tpac文書の生成に失敗したことを表す例外クラスです。
    TpacSyntaxException tpac文書の文法誤りを表す例外クラスです。
